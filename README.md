## GNOME Shell Extensions

This image includes pre-installed GNOME Shell extensions via git submodules:

- **AppIndicator Support** (gnome-45) - System tray icon support
- **Blur My Shell** (master) - Adds blur effects to GNOME Shell
- **Caffeine** (master) - Prevents screen from dimming/locking
- **Dash to Dock** (ubuntu-dock) - Customizable dock for GNOME
- **GSConnect** (main) - KDE Connect integration for GNOME
- **Logo Menu** - Universal Blue logo menu integration
- **Search Light** (gnome-48) - Enhanced search functionality âœ¨

Extensions are built from source during image creation and have extension version validation disabled to ensure compatibility.
